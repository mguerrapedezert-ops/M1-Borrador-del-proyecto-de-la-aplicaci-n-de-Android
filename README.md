# Uruguayn’t

## Descripción del proyecto
Uruguayn’t es una aplicación para estudiantes que buscan organizar mejor su día y evitar dejar todo para último momento. La app analiza horarios, identifica tiempos libres y propone un plan diario equilibrado.

## Exposición del problema
Muchos estudiantes tienen dificultades para administrar horarios y terminan postergando sus responsabilidades. La falta de organización afecta el rendimiento y genera estrés. Esta app ordena el día y distribuye tareas automáticamente.

## Plataforma
Aplicación desarrollada en Android Studio (Kotlin + XML).

## Interfaz de usuario
- Inicio y registro
- Pantalla de “Conocerte”
- Agenda diaria con sugerencias de estudio
- Estadísticas personales
- Notificaciones de actividades

## Funcionalidad
- Crear cuenta
- Registrar horarios y rutinas
- Sugerencias automáticas
- Notificaciones
- Panel de estadísticas

2. Conocerte
